# Rolin Bio 网站部署说明

## 域名

- **域名：** rolinbio.com
- **网站目录：** `rolinbio-website/`

---

## 方案一：Netlify 部署（推荐，支持 Decap CMS）

### 步骤

1. **注册/登录 Netlify**
   访问 https://app.netlify.com/

2. **连接 Git 仓库**
   - 将 `rolinbio-website/` 目录初始化为 Git 仓库并推送到 GitHub/GitLab
   - 在 Netlify 中选择 "Import from Git"

3. **配置构建设置**
   - Build command: （留空，纯静态站点）
   - Publish directory: `.`

4. **配置域名**
   - Netlify Dashboard → Domain settings → Add custom domain
   - 添加 `rolinbio.com`
   - 更新域名 DNS：添加 CNAME 记录指向 Netlify 提供的 URL

5. **启用 Netlify Identity（CMS 后台必需）**
   - Netlify Dashboard → Identity → Enable Identity
   - Registration → Invite only（推荐）
   - 邀请管理员邮箱

6. **启用 Git Gateway**
   - Netlify Dashboard → Identity → Services → Enable Git Gateway

7. **访问 CMS 后台**
   https://rolinbio.com/admin/

---

## 方案二：静态托管（无需 CMS）

直接将 `rolinbio-website/` 目录上传到任意静态托管服务：

- **CloudStudio**（腾讯云）
- **GitHub Pages**
- **阿里云 OSS**
- **Vercel**

如果不需要 CMS 后台，删除 `admin/` 目录即可。

## 新闻内容更新方式

### 方法 A：通过 CMS 后台（推荐）

1. 访问 https://rolinbio.com/admin/
2. 用 Netlify Identity 账户登录
3. 在 "News & Insights" 集合中创建/编辑文章
4. 保存后自动提交到 Git 仓库并触发 Netlify 部署

### 方法 B：手动编辑 Markdown

1. 在 `content/news/` 目录下创建 `.md` 文件
2. 格式参考已有示例文件
3. 提交并推送到 Git 仓库

```markdown
---
title: "文章标题"
date: 2026-06-27
category: "regulatory"
excerpt: "文章摘要，150字以内"
image: ""
---

文章正文（Markdown 格式）...
```

## 自定义修改

### 公司信息
- 联系方式：编辑 `contact.html`
- 公司介绍：编辑 `about.html`
- 业务描述：编辑 `services.html`

### 颜色/样式
- 全局样式变量在 `assets/css/style.css` 的 `:root` 块中

### Google Fonts
- 字体加载在各 HTML 页面的 `<head>` 中
- 如需离线使用，下载字体文件放入 `assets/fonts/`
